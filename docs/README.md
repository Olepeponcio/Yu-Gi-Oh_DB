# Documentacion del proyecto

Indice de documentos tecnicos del proyecto.

## Documentos

- `api_json_analysis.md`: analisis del JSON de YGOPRODeck y entidades candidatas.
- `etl_flow.md`: flujo API -> raw JSON -> transformacion -> MySQL.
- `data_model.md`: tablas, relaciones y criterios de modelado.
- `sql_usage.md`: uso de scripts SQL y consultas de validacion.

## Criterio de organizacion

- `README.md` en raiz: guia rapida del proyecto.
- `docs/`: explicaciones tecnicas y decisiones reutilizables.
- `sql/`: scripts ejecutables en MySQL.
- `src/`: codigo Python del ETL.
